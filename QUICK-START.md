## 🔧 Troubleshooting

**Images not showing?**
1. Hard refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
2. Check server is running: `npm run dev`
3. Clear browser cache

**Still issues?**
Check browser console (F12) for errors.
